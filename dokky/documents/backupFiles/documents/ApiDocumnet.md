## 1) 글 작성

### HTTP Request Header

HTTP Header| value 
---- | ----
Method | POST
URI Path | /board/register

### HTTP Request Body

key | comment | type | 필수
---- | ---- | ---- | ----
category | 글 카테고리 종류 번호(0~5) | Int | O
title | 글 제목 | String | O
nickName | 작성자 닉네임 | String | O
userId | 작성자 아이디 | String | O
content | 글 내용 | String | O

### Request Body example

key | value 
---- | ----
category | 1   
title | 공지사항 입니다.
nickName | 슈퍼 관리자 
userId | admin 
content | 도키 커뮤니티에 오신것을 환영합니다.
